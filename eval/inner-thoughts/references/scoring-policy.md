# Scoring Policy

Score the run by effect and contract, not exact wording.

Use 1-5 scores for:

- `entry_creation`: home composer created a real session and saved the first user message.
- `reflection_quality`: MWF helped the actor think without rushing or sounding generic.
- `cta_judgment`: partner-session CTA appeared when earned and stayed absent when not earned.
- `handoff_quality`: CTA/new-session/Stage 0 carried name and context correctly.
- `ui_quality`: CTA, input, keyboard, safe area, and drawer flow were polished and non-overlapping.
- `actor_fidelity`: actor followed scenario posture without forcing outcomes.

Every score below 4 must name owner, evidence, and recommended action.
