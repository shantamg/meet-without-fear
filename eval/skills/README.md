# MWF Gold Skills

This directory keeps git-versioned copies of the local Codex skills used for gold-session work.

- `manual/`: skills intended for hands-on Codex Desktop runs with the in-app browser.
- `self-improvement/`: skills intended for `scripts/mwf_gold_loop.py` and command-line automation.

Installed runtime copies live in `/Users/shantam/.codex/skills/`. After editing a versioned skill here, run:

```bash
scripts/sync_mwf_gold_skills.sh
```
